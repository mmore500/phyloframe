## Citing

If you use phyloframe in your research, please cite it via the GitHub repository at <https://github.com/mmore500/phyloframe>.

bibtex:
```bibtex
@software{moreno2024phyloframe,
  author = {Matthew Andres Moreno},
  title = {phyloframe: phylogenetic data frame tools},
  url = {https://github.com/mmore500/phyloframe},
  year = {2024},
}
```
